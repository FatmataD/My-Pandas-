# My-Pandas-
# 🐼📊 Pandas Data Analysis Project – GDP Per Capita

## 📌 Overview
This project was completed as part of a Data Technician bootcamp and focuses on using the pandas library in Python to explore, clean, and analyze retail and sales data. The objective was to transform raw datasets into structured, meaningful insights through efficient data manipulation and visualization techniques.

---

## 🛠️ Skills & Techniques Demonstrated

### 🧮 Data Manipulation with Pandas
- Used **pandas** for efficient data handling and analysis  
- Applied filtering techniques to isolate relevant subsets of data  
- Performed subsetting using:
  - `.loc[]` for label-based indexing  
  - `.iloc[]` for position-based indexing  

---

### 🔄 Data Cleaning & Preparation
- Managed missing data using:
  - `dropna()` to remove null values  
  - `fillna()` to replace missing values appropriately  
- Ensured datasets were clean, consistent, and ready for analysis  

---

### 📊 Data Analysis
- Leveraged pandas methods to summarize and analyze data:
  - `groupby()` to aggregate metrics across categories  
  - `sort_values()` to rank and organize results  
- Explored key business questions such as:
  - Which products generate the most revenue?  
  - How do sales vary by region or time period?  

---

### 📈 Data Visualisation
- Created visualizations using:
  - Built-in pandas plotting functionality  
  - **matplotlib** for enhanced customization  
- Developed charts including:
  - **Bar charts** for category comparisons  
  - **Line charts** for trend analysis  
- Used visuals to clearly communicate insights  

---

## 📖 Data Storytelling
- Transformed raw retail data into **clear, actionable insights**  
- Highlighted trends, patterns, and anomalies through visual analysis  
- Presented findings in a way that supports data-driven decision-making  

---

## 📈 Key Outcomes
- Cleaned and structured real-world retail datasets  
- Identified trends and performance patterns in sales data  
- Strengthened data analysis and visualization skills using pandas
